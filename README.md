# Software Engineering Project

Reports and documentation for software engineering project at the university of Leipzig during summer term 2026.

## Dependencies
- [typst](https://typst.app/)
- [just](https://github.com/casey/just) (optional, for task running)
- [pympress](https://github.com/Cimbali/pympress) (optional, for presenting)

## Usage
Compile everything (requires just)
```sh
just
```

Start the presenation (requires just and pympress)
```sh
just present
```
